Spectrum 2 moves Adobe's product surfaces toward lighter, bolder, rounder components, while the marketing language stays disciplined: white surfaces, near-black text, and red used only as a rare, deliberate accent.

Motion in this kit follows a strict hierarchy — small component-level transitions borrow Spectrum's own duration and easing tokens, while larger marketing-weight reveals use a stronger custom curve so they read with enough presence at scale.

Every interactive control stays under the 300ms UI ceiling; only overlays, trays, and scroll-triggered choreography are permitted to run longer, and only because the user isn't actively waiting on direct feedback from them.

Reduced-motion users get every transition collapsed to a near-instant one millisecond rather than deleted outright, so dependent event listeners keep firing correctly while movement effectively disappears.

This progress bar is a direct, unsmoothed mapping from scroll position to bar width — no easing curve is layered on top, since scrubbed motion like this should track the input with zero perceptible lag.
